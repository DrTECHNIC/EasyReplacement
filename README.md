# Дешифровка текста методом простой замены

## Описание
Это приложение предназначено для дешифровки текста, зашифрованного методом простой замены символов. Оно позволяет вводить зашифрованный текст и постепенно подбирать замены для символов, наблюдая за результатом дешифровки в реальном времени.

## Установка и запуск

### Запуск из исходного кода
1. Убедитесь, что у вас установлен Python 3.6 или выше
2. Склонируйте репозиторий с GitHub: `git clone https://github.com/DrTECHNIC/EasyReplacement.git`
3. Переместитесь в папку проекта: `cd EasyReplacement`
4. Запустите приложение командой: `python main.py`

### Сборка исполняемого файла
1. Убедитесь, что у вас установлен Python 3.6 или выше
2. Запустите скрипт сборки: `python build.py`
3. После завершения сборки в текущей папке появится файл `cipher_tool.exe`

## Руководство пользователя

### Основные элементы интерфейса

1. **Поле "Зашифрованный текст"**
- Введите или вставьте зашифрованный текст в это поле
- Текст автоматически преобразуется к верхнему регистру

2. **Поле "Текст для дешифровки"**
- Отображает результат дешифровки в реальном времени
- Верхняя строка каждого блока показывает дешифрованные символы
- Нижняя строка показывает оригинальные зашифрованные символы
- Символы, для которых еще не задана замена, отображаются как подчеркивания (`_`)

3. **Таблица замен**
- Содержит все буквы русского алфавита (без Ё, Й, Ъ)
- Для каждой буквы можно указать соответствующую дешифрованную букву
- Поля ввода активны только для букв, присутствующих в зашифрованном тексте

### Процесс дешифровки

1. **Ввод текста**
- Вставьте или введите зашифрованный текст в верхнее поле
- Текст автоматически разбивается на строки по 75 символов и группы по 5 символов

2. **Анализ текста**
- Изучите частоту появления символов и возможные паттерны
- Обратите внимание на часто встречающиеся символы - это могут быть гласные или распространенные согласные

3. **Задание подстановок**
- В таблице замен для каждого символа введите предполагаемую дешифрованную букву
- Можно вводить только буквы русского алфавита (без Ё, Й, Ъ)
- Для очистки подстановки удалите символ из поля ввода

4. **Наблюдение за результатом**
- По мере ввода подстановок текст в области дешифровки будет автоматически обновляться
- Сравнивайте оригинальные и дешифрованные символы для контроля правильности

### Советы по дешифровке

1. Начните с самых частых символов - в русском языке это обычно гласные (А, Е, И, О, У)
2. Ищите короткие слова (предлоги, союзы) которые могут помочь определить структуру текста
3. Обращайте внимание на повторяющиеся паттерны - это могут быть окончания слов или корни
4. Используйте знания о частотности букв в русском языке

### Особенности приложения

- Приложение работает только с буквами русского алфавита (без Ё, Й, Ъ)
- Все символы автоматически преобразуются к верхнему регистру
- Пробелы сохраняются в оригинальном виде
- Подстановки применяются мгновенно и отображаются в реальном времени
- Размер окна фиксирован для оптимального отображения всех элементов

## Техническая информация

### Поддерживаемые символы
Приложение поддерживает следующие символы русского алфавита:  
А Б В Г Д Е Ж З И К Л М Н О П Р С Т У Ф Х Ц Ч Ш Щ Ы Ь Э Ю Я

### Формат текста
- Текст разбивается на строки по 75 символов
- Каждая строка разбивается на группы по 5 символов
- Между группами вставляются пробелы для лучшей читаемости

## Разработка

### Структура проекта
- `main.py` - точка входа приложения
- `interface.py` - графический интерфейс пользователя
- `decoder.py` - логика дешифровки текста
- `build.py` - скрипт для сборки исполняемого файла
- `public/icon.png` - иконка приложения

### Сборка
Для сборки исполняемого файла используется PyInstaller. Запустите `python build.py` для автоматической сборки.
